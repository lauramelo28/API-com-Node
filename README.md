Implementando uma API em Node.Js e persistência no SqlServer Express, que possibilita salvar os seguintes dados de cadastro de produtos:

* Nome do produto;
* Código do produto;
* Preço;
* Descrição;
* Quantidade em estoque;
* Avaliação; 
* Categoria.

Utiliza as ações do CRUD para criar, ler, editar e deletar dados do banco SqlServer Express. Desta forma, a sua API deve ter rotas para:

* Criação de dados com Post;
* Leitura de todos os dados do banco com Get;
* Leitura de dados por código do produto com Get;
* Leitura de dados por categoria com Get;
* Atualização de informações com Put ou Patch; e
* Exclusão de elementos com Delete.
